Model notes / glossary
Quick definitions of terms used in this report and how the main adjustments behave.
What this is
Daily win probabilities + model-implied moneylines, with team-profile radars and a frozen-accuracy backtest.
Win probability
Model-estimated chance of the predicted winner winning the game.
Model ML
Moneyline derived from the model probability (no vigorish).
ESPN ML
Market moneyline pulled from ESPN (may include vig).
Devig
If both sides are available, market implied probs are normalized so home+away=1.
Confidence buckets
Very High ≥ 72% · High ≥ 65% · Medium ≥ 58% · Low ≥ 50% · otherwise N/A.
Frozen predictions
Predictions are snapshotted into prediction_history.csv when games become upcoming; accuracy uses only those frozen rows.
Radars
Each spoke is a team percentile vs league for that metric (higher = stronger). Filled areas use secondary team color.
Radar metrics (FEATURES)
• xGoals% (share of expected goals) (xGoalsPercentage)
• Fenwick% (unblocked shot attempt share) (fenwickPercentage)
• xGoals For / 60 (xGoalsForPer60)
• xGoals Diff / 60 (for − against) (xGoalsDiffPer60)
• Shots Diff / 60 (SOG for − against) (shotsDiffPer60)
• High-Danger Diff / 60 (highDangerDiffPer60)
• Shot Credit Diff (score-adjusted) (shotCreditDiff)
• Penalty Diff (for − against) (penaltyDiff)
• Goal Diff (for − against) (goalDiff)
• Save% (approx; SOG saved / SOG against) (savePct)
xG shot-form
Optional adjustment from MoneyPuck shots: rolling window=10 games, scaled by 0.35, capped at 0.75.
Recency form
Margin-weighted recent form over last 10 games; blended with weight 0.3.
Rest / fatigue
Rest differential term with weight 0.1 and clip ±3 days.
Home/road split
Home-vs-road performance delta with weight 0.2 and prior=20 games.
What to watch
Higher confidence ≠ guarantee; look for injuries/goalies/news that the model doesn’t fully capture.
Model-driven probabilities — not betting advice.
Accuracy uses frozen predictions from prediction_history.csv and completed games only.
